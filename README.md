## 🚀 Features

- ✅ Built with **Discord.js v14**
- 🌐 **Multi-language support** with database
- 🧠 **Custom Client structure**
- ⚙️ **Event Handler**
- 💻 **Advanced Slash Command Handler**
- 🛠️ **Developer Guild-only Commands**

---

## 💫 Get Started

Install dependencies and start your bot:

```bash
npm install
npm run start
```
